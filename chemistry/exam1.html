Q1
平衡係數的目的是符合哪個定律？
定比 / 倍比 / 原子說 / 分子說
(C) 原子說
▾
達爾頓的「原子說」：原子不會憑空出現，也不會憑空消失。
平衡係數讓原子個數守恆，符合原子說。
選項定義是否符合
定比定律同一化合物各元素質量比固定✗ 描述化合物性質
倍比定律兩種元素形成多種化合物，質量比為簡單整數比✗ 描述多種化合物
原子說原子不滅 → 方程式兩邊原子總數相等✓ 正確
分子說描述物質由分子組成✗ 不描述反應
Q2
化學方程式撰寫原則，甲乙丙丁哪些正確？
甲：用「=」連接 / 乙：條件標箭頭上方 / 丙：係數非最小代表反應不存在 / 丁：分子量與係數無關
(D) 乙、丁
▾
選項說法正確？原因
甲反應物與生成物用「=」連接✗應用「→」，不是等號
乙反應條件可標在箭頭上方✓加熱、催化劑等條件寫在箭頭上方是正確寫法
丙係數不是最小整數代表反應不存在✗只代表可以再化簡，反應依然存在
丁分子量與係數大小無關✓分子量是固定值，係數不改變化學式本身
Q3
關於化學反應方程式的敘述，下列何者正確？
判斷四個說法的對錯
(C)
▾
說法正確？原因
A化學反應式可視為化學式的簡稱，意義完全相同✗化學式只描述組成，反應方程式描述反應過程
B為方便計算可以憑空杜撰係數✗係數必須依照原子守恆決定，不能亂寫
C分子種類一定改變，但原子個數必須相等✓反應 = 生成新分子（種類變），但原子不滅（個數不變）
D「→」代表反應的快慢程度✗「→」代表反應方向，不代表快慢
Q4
哪些性質在化學反應前後「一定會改變」？
甲乙丙丁戊己 六個性質
(B) 丙、丁
▾
性質結果理由
甲原子的種類不一定原子種類守恆
乙原子的總個數不改變質量守恆 → 原子個數守恆
丙原子的排列方式一定改變 ✓化學反應就是原子重新排列
丁分子的種類一定改變 ✓生成新物質，分子種類必然不同
戊分子的總質量不改變質量守恆
己分子個數不一定N₂+3H₂→2NH₃：4個→2個；H₂+Cl₂→2HCl：2個→2個
只有「丙」和「丁」是一定改變的！己（分子個數）不一定改變，所以選 (B) 丙、丁，不選 (C) 丙、丁、己。
Q5
以下哪個關於質量的描述是正確的？
雙氧水 / 蠟燭 / 鎂 / 氧化汞 四種情境
(A)
▾
情境正確？解析
A10g 雙氧水分解，所得「水+氧」的質量 = 10g✓ 正確質量守恆：水 + 氧 = 原本的雙氧水
B10g 蠟燭燃燒，所得「水+CO₂」= 10g✗ 錯燃燒需要吸收空氣中的O₂，產物 > 10g
C10g 鎂在空氣燃燒，MgO質量 = 10g✗ 錯Mg + O₂ → MgO，MgO > 10g（加上O的質量）
D10g 氧化汞分解，所得汞 = 10g✗ 錯HgO → Hg + O₂，O₂跑走了，汞 < 10g
關鍵：A 問的是「水和氧合起來」的質量。2H₂O₂ → 2H₂O + O₂，水 + 氧的總質量 = 原本雙氧水（守恆）。

Q19
MnO₂ 在甲反應和乙反應中分別是什麼角色？
催化劑 vs 反應物
(C)
▾
催化劑的特徵：參與反應、加速反應，但反應前後本身不被消耗
甲反應：4HCl + MnO₂ → Cl₂ + MnCl₂ + 2H₂OMnO₂ 變成了 MnCl₂ → 被消耗 → 反應物
乙反應：2H₂O₂ —(MnO₂)→ 2H₂O + O₂MnO₂ 寫在箭頭上方 → 未被消耗 → 催化劑
反應MnO₂ 的角色判斷依據
甲反應物出現在反應式左邊且被消耗（變成MnCl₂）
乙催化劑寫在箭頭上方，反應前後不變

▾

(NH₄)₂CO₃ → CO₂ + H₂O + 2NH₃驗算：N:2=2✓ H:8=2+6✓ C:1=1✓ O:3=2+1✓

Q9
2A₂ + B₂ → 2A₂B，哪個比值「不等於」2:1:2？
分子數比 / 莫耳比 / 體積比 / 質量比
(D) 質量比
▾
比較項目等於2:1:2？原因
莫耳數比= 2:1:2直接看係數就是莫耳比
分子數比= 2:1:2分子數 = 莫耳數 × 亞佛加厥數，比值不變
氣體體積比（同溫同壓）= 2:1:2同溫同壓下體積∝莫耳數
質量比≠ 2:1:2質量 = 莫耳數 × 分子量，要再乘分子量才行
質量比 ≠ 係數比！例：2mol A₂ : 1mol B₂ 的質量比 = 2×M(A₂) : 1×M(B₂)，不一定是2:1

Q29
