Concept
觀念判斷
Q1–6、Q18–20 · 不需計算，考的是對化學反應本質的理解
解題核心
  • ①
    一定改變的只有兩個：原子排列方式、分子種類
    不改變：原子種類、原子個數、物質總質量
    不一定：分子個數
  • ②
    催化劑判斷：反應前後自身不消耗 = 催化劑（寫在箭頭上方）；出現在左側且消失 = 反應物
  • ③
    反推未知物：先數左邊各原子總數，減去右邊已知物用掉的，差額就是未知物的組成
  • ④
    開放容器質量下降 ≠ 違反守恆：有氣體逸出，所以容器內質量減少，整體原子質量守恆
Q4 最常錯：「分子個數」不是一定改變。
反例：H₂ + Cl₂ → 2HCl — 2個分子反應後還是2個，個數沒變。
段考精華題
Q19
MnO₂ 在甲反應（製Cl₂）和乙反應（製O₂）中各是什麼角色？
催化劑 vs 反應物的判斷
答 C
▾
甲：4HCl + MnO₂ → Cl₂ + MnCl₂ + 2H₂OMnO₂ 變成 MnCl₂ → 被消耗 → 反應物
乙：2H₂O₂ —MnO₂→ 2H₂O + O₂寫在箭頭上方 → 未被消耗 → 催化劑
同樣一個物質，在不同反應裡角色可以完全不同。判斷依據永遠是：它有沒有被消耗？
Q5
10g 鎂在空氣中完全燃燒，所得 MgO 質量為 10g？
開放容器的質量邏輯
答 A
▾
C 選項「MgO = 10g」是錯的。Mg + O₂ → MgO，O₂ 從空氣中加進來，產物 MgO 比原本的 Mg 重。
A 選項正確邏輯：問的是「水和氧合起來的質量」= 原本 H₂O₂ 的質量，這才是守恆的那一側。
開放容器情境質量變化 有固體/液體從空氣中進來（如 O₂）↑ 增加 有氣體逸出（如 CO₂、O₂ 逸出）↓ 減少 只有沉澱或溶液，無氣體幾乎不變
Q15
(NH₄)₂CO₃ → CO₂ + H₂O + 2X，X 是什麼？
由產物組成反推未知物
答 C
▾
1
左邊原子：N=2, H=8, C=1, O=3
2
CO₂ + H₂O 用掉：C=1, O=3, H=2 → 還剩 N=2, H=6 未分配
3
2X 要帶走 N=2, H=6 → 每個 X = N₁H₃ → X = NH₃（氨氣）
延伸練習
延
下列哪個一定是「物理變化」？(A) 蠟燭燃燒 (B) 乾冰昇華 (C) 鐵生鏽 (D) 食物腐敗
▾
答 (B) 乾冰昇華 = CO₂ 固→氣，狀態改變，分子種類沒變 → 物理變化。其餘都生成新物質。
延
催化劑的添加量增加，下列何者正確？(A) 生成物增加 (B) 反應速率加快 (C) 生成物種類改變 (D) 消耗的反應物增加
▾
答 (B) 催化劑只影響反應速率，不影響產物種類和數量。加更多催化劑 → 反應更快，但最終產物一樣多。
延
C₂H₅OH + 3O₂ → 2CO₂ + 3H₂O，反應前後「分子個數」變化？
▾
反應前：1 + 3 = 4 個分子；反應後：2 + 3 = 5 個分子
→ 分子個數從 4 → 5，增加了。這就是「分子個數不一定改變」的延伸案例。
延
A₂B + 3C → 2AC + ½B₂，B₂ 係數是分數。下列敘述何者正確？
(A) 此反應不存在 (B) 應全體×2化為整數 (C) 係數可以是分數沒關係 (D) B₂係數應該改成1
▾
答 (B) 係數必須是最簡整數。出現分數就全體乘以分母：×2 得 2A₂B + 6C → 4AC + B₂。
反應依然存在，只是沒化到最簡。
延
某反應：A + B₂ → AB₂ + 熱，在密閉容器中進行。反應後容器「重量」與「溫度」各有何變化？
▾
密閉容器 → 沒有物質進出 → 重量不變（質量守恆）
有「+熱」產生 → 溫度上升（放熱反應）
質量守恆與能量無關，放熱只讓溫度升，不讓質量消失。
預測重點
預
吸熱反應 vs 放熱反應 — 如何判斷？
反應過程中溫度上升還是下降？
▾
類型溫度變化典型例子
放熱反應↑ 升高燃燒、石灰加水（CaO+H₂O）、金屬與酸
吸熱反應↓ 降低碳酸鈣加熱分解、光合作用、電解
若題目說「反應後溫度升高」→ 放熱反應。在開放容器中，放熱反應不影響質量，只影響溫度。
「加熱才能進行」的反應是吸熱（需要輸入能量維持反應）。
預
氧化還原基礎 — 加氧 vs 失氧
國中版本只看「氧」，不需要看電子轉移
▾
概念定義例子
氧化得到氧（被加氧）2Mg + O₂ → 2MgO（Mg被氧化）
還原失去氧（被去氧）Fe₂O₃ + 3C → 4Fe + 3CO₂（Fe₂O₃被還原）
氧化劑提供氧給別人上例中的 Fe₂O₃
還原劑奪走別人的氧上例中的 C
燃燒 = 可燃物與 O₂ 劇烈氧化，可燃物是還原劑，O₂ 是氧化劑。
煉鐵 = 焦炭（C）奪走 Fe₂O₃ 的氧 → Fe₂O₃ 被還原成 Fe。
預
化學方程式的「量」有三層意義，下列何者正確？
以 2H₂ + O₂ → 2H₂O 為例
▾
層次H₂O₂H₂O
分子個數比2個1個2個
莫耳數比2mol1mol2mol
質量（g）4g32g36g
個數比 = 莫耳比 = 體積比（同溫同壓氣體）
但質量不等比例，必須分別乘分子量計算。
驗算：4+32=36g ✓（質量守恆）
預
酸雨的成因 — SO₂ 和 NOₓ 如何變成酸？寫出相關方程式
生活情境 + 化學方程式撰寫
▾
SO₂ + H₂O → H₂SO₃（亞硫酸）手寫題一(2) 的原理來源，化合反應
2SO₂ + O₂ → 2SO₃ → SO₃ + H₂O → H₂SO₄工廠排放 SO₂ → 空氣中氧化 → 硫酸 → 酸雨
酸雨的 pH 小於 5.6。SO₂ 主要來自燃煤發電廠和火山；NO₂ 來自汽車廢氣。兩者都先溶於水再形成酸。
預
下列各種變化，哪些屬於「化學變化」？哪些是「物理變化」？
水結冰 / 紙張燃燒 / 食鹽溶水 / 牛奶發酸 / 鐵塊被磁鐵吸引 / 鋼鐵生鏽
▾
變化類型判斷依據
水結冰物理H₂O 液→固，分子種類未變
紙張燃燒化學生成 CO₂、H₂O，新物質
食鹽溶水物理NaCl 只是分散，仍是 NaCl
牛奶發酸化學乳酸發酵，生成新物質
鐵塊被磁鐵吸引物理無新物質，只是位移
鋼鐵生鏽化學Fe + O₂ + H₂O → Fe₂O₃，生成新物質
Balance
平衡方程式
Q15–17、Q23 · 手寫一、二、四 · 讓兩邊每種原子個數相等
設 1 法（萬用步驟）
  • ①
    最複雜的物質係數設為 1（原子種類最多，或含括號的那個）
  • ②
    平衡順序：金屬 → C → H → 最後才處理 OO₂ 一次帶兩個，留到最後用分數解，再統一乘倍
  • ③
    出現分數 → 全體乘以分母，消分數後再驗算
  • ④
    驗算：每種原子逐一清點兩邊容易漏：NO₃⁻ 裡有 3 個 O；Ca(OH)₂ 裡有 2 個 OH
四種反應類型
  • 化合
    多→一（A + B → AB）4Na + O₂ → 2Na₂O
  • 分解
    一→多（AB → A + B）CaCO₃ → CaO + CO₂
  • 取代
    A 踢走 B（A + BC → AC + B），活性強的元素取代活性弱的2Al + 6HCl → 2AlCl₃ + 3H₂
  • 複分解
    兩對換伴（AB + CD → AD + CB）AgNO₃ + NaCl → AgCl↓ + NaNO₃
最常犯的錯：把 H₂O 的下標改成 H₄O → 這樣化學式就錯了！
正確做法：只能在前面加係數 2H₂O，不能改化學式本身。

預測重點
預
電解水：2H₂O → 2H₂ + O₂，電解 36g 水，生成 H₂ 和 O₂ 各多少克？
電解 = 分解反應；H₂:O₂ 體積比 = 2:1
▾
2H₂O → 2H₂ + O₂質量比 H₂O:H₂:O₂ = 36:4:32 = 9:1:8
1
36g H₂O = 2mol → 生成 2mol H₂ = 4g，1mol O₂ = 32g
2
驗算：4 + 32 = 36g ✓（質量守恆）
3
同溫同壓下，H₂:O₂ 的體積比 = 2:1（莫耳比）
常考變化：問「體積比」而非質量比 → 用莫耳比2:1；問「質量比」→ 4:32 = 1:8。
預
2KClO₃ →(MnO₂, 加熱) 2KCl + 3O₂，平衡並判斷MnO₂角色，收集到48g O₂需分解多少KClO₃？
KClO₃=122.5，O₂=32
▾
2KClO₃ —(MnO₂, △)→ 2KCl + 3O₂分解反應；MnO₂ 在箭頭上方 → 催化劑
1
48g O₂ = 1.5mol；KClO₃:O₂ = 2:3 → KClO₃ = 1mol
2
1mol × 122.5 = 122.5g KClO₃
預
平衡 C₂H₂ + O₂ → CO₂ + H₂O（乙炔燃燒），並與甲烷 CH₄ 比較每莫耳燃燒的耗 O₂ 量
▾
2C₂H₂ + 5O₂ → 4CO₂ + 2H₂O每mol C₂H₂ 耗 O₂ = 2.5mol
CH₄ + 2O₂ → CO₂ + 2H₂O每mol CH₄ 耗 O₂ = 2mol
C₂H₂ 的 H:C 比低（1:1），含氫量相對少，因此相比 CH₄（H:C=4:1）耗 O₂ 更多。
乙炔是焊接用的高溫燃料，燃燒溫度超過 3000°C。

平衡 NH₃ + O₂ → NO + H₂O（工業製硝酸的第一步），求係數總和
▾
1
設 NH₃=1 → N→NO係數=1，H=3→H₂O=3/2（有分數）
2
O：右邊 1+3/2=5/2 → O₂=5/4，全體×4
3
4NH₃ + 5O₂ → 4NO + 6H₂O，係數總和 = 4+5+4+6 = 19
Mass
質量守恆計算
Q7–8、Q10、Q21–22、Q24、Q26、Q28
兩種情境，一個邏輯
  • 密閉
    反應前總質量 = 反應後總質量，直接加減求未知Q8：前 6+58+5+23=92g，後 X+10+32=92 → X=50g
  • 開放
    逸散氣體 = 反應前容器重 − 反應後容器重Q10：360−344=16g O₂ 逸出
  • 比例
    係數×分子量 = 質量基準；再用比例式求未知Q7：CaCO₃:CO₂=100:44，求176g CO₂ 需要 x/100=176/44 → x=400g
  • 圖形
    質量-時間曲線下降 → 有氣體逸出；斜率 = (原料+加入物)/原料Q26：斜率=5/4，即4g Cu對應1g O₂

Q28
甲金屬24.3g 與乙金屬65.4g 各與鹽酸反應，都產生2g H₂，鹽酸用量和產物質量是否相同？

▾

分別取質量不等的鐵粉在空氣中加熱至全黑，從「加熱前質量」對「加熱後質量」的圖，斜率代表什麼？
▾

▾

A 消耗 = 3−0.5 = 2.5mol
2

3

預測重點

Q9、Q11–14、Q25、Q27、Q29–30 · 係數比 = 莫耳比，是整章核心
三個核心公式
  • ①
    莫耳數 = 質量 ÷ 分子量
  • ②
    係數比 = 莫耳比 = 氣體體積比（同溫同壓）但不等於質量比！質量比要各乘分子量
  • ③
    限量試劑：比較「有多少」vs「需要多少」，不夠的那個決定產物量
解題流程（5步）
  • 1
    寫出平衡方程式
  • 2
    所有質量÷分子量，換成莫耳數
  • 3
    若有兩種反應物，先找限量試劑
  • 4
    用係數比求產物莫耳數
  • 5
    若問克數，再乘分子量
