觀念判斷

測試對化學反應本質的理解，不需計算

Q1Q2Q3Q4Q5Q6Q18Q19Q20
解題要訣
  • 1
    反應前後「一定改變」的只有兩件事：原子的排列方式、分子的種類 原子種類／個數／總質量都不改變。分子個數「不一定」改變。
  • 2
    催化劑的判斷：反應前後自身不被消耗（重量不變），只加快速率 出現在反應式箭頭上方 = 催化劑；出現在左側且被消耗 = 反應物
  • 3
    觀察化學變化的方法要配合反應產物：有氣泡逸出 → 選有氣體生成的反應 只生成沉澱或溶液的反應，不適合用「觀察氣泡」判斷
  • 4
    反推未知物質：先數左邊各原子個數，再數右邊已知物質用掉多少，差額就是未知物帶走的 Q15：(NH₄)₂CO₃ 分解，CO₂ 和 H₂O 用掉部分原子，剩下 N + H → NH₃
常見陷阱
「分子個數」不一定改變 — Q4 最容易誤選「己（分子個數）也是一定改變」。
反例：H₂ + Cl₂ → 2HCl，2個分子反應後還是2個分子，個數沒變。
段考題型 vs 變化題型
段考考法變化題型
反應前後哪些性質改變（Q4） 改成問「哪些一定不改變」或「哪些可能改變」
MnO₂ 在兩個反應中的角色（Q19） 換成 Pt 或 Fe 作催化劑，問同樣概念；或問催化劑的用量對反應的影響
指定反應物推算未知 X（Q15） 給完整產物，倒推某個反應物是什麼分子

暖暖包的主要發熱反應，化學方程式為何？ ▼
1

平衡方程式

讓兩邊每種原子的個數相等，係數取最簡整數

Q15Q16Q17Q23 手寫一手寫二手寫四
解題要訣 — 設 1 法
  • 1
    把最複雜的物質係數設為 1（原子種類最多、下標最大的那個） 通常是有機物或多原子分子，例如 C₈H₁₈、Cu(NO₃)₂
  • 2
    平衡順序：金屬 → C → H → 最後才處理 O O 最難，因為 O₂ 一次帶兩個，留到最後用分數處理再乘倍
  • 3
    出現分數時全部係數乘以分母 ½O₂ → 全部 ×2，½ 消失，所有係數變整數
  • 4
    驗算：逐種原子數清點，兩邊都對才算完成 常見漏掉 NO₃⁻ 裡的 O，或 Ca(OH)₂ 裡有兩個 OH
段考重點：係數總和
Q17 問「係數總和」→ 平衡後把所有數字加起來。
3Cu + 8HNO₃ → 3Cu(NO₃)₂ + 4H₂O + 2NO
3 + 8 + 3 + 4 + 2 = 20
常見陷阱
只能改係數，不能改化學式！
✗ 錯誤：把 H₂O 改成 H₄O 讓 H 多出來
✓ 正確：在 H₂O 前面加係數 2，寫成 2H₂O
四種反應類型 — 快速辨認
類型形式記憶法例子
化合A + B → AB多→一（合體）Na + O₂ → Na₂O
分解AB → A + B一→多（分裂）CaCO₃ → CaO + CO₂
取代A + BC → AC + BA 踢走 BAl + HCl → AlCl₃ + H₂
複分解AB + CD → AD + CB兩對換伴AgNO₃ + NaCl → AgCl + NaNO₃
補充變化題 — 練習看看
平衡並求係數總和：C₂H₅OH + O₂ → CO₂ + H₂O（酒精燃燒） ▼
1
設 C₂H₅OH = 1，C → CO₂ 係數 = 2，H → H₂O 係數 = 3
2
O：右邊 4+3=7，左邊 1（來自C₂H₅OH）+O₂：需補 6/2=3 個 O₂
3
C₂H₅OH + 3O₂ → 2CO₂ + 3H₂O，係數總和 = 1+3+2+3 = 9
下列哪個敘述正確？(A) 係數越大反應越快 (B) 係數不是最小整數代表反應不存在 (C) 反應條件寫在箭頭上方 (D) 反應物與生成物用「=」連接 ▼
答 (C)

(A) 係數只表示莫耳比，與反應速率無關
(B) 係數可以化簡，反應依然存在（Q2 段考考點）
(C) ✓ 加熱、催化劑等條件標在箭頭上方
(D) 應用「→」不是「=」
平衡 Fe₃O₄ + CO → Fe + CO₂（工業煉鐵變化版），求係數最小整數比 ▼
1
設 Fe₃O₄ = 1 → Fe 係數 = 3
2
O：左邊 4，CO₂ 中的 O 只來自 CO，CO 消耗 = CO₂ = 4
3
Fe₃O₄ + 4CO → 3Fe + 4CO₂，係數比 1:4:3:4
03

質量守恆計算

密閉容器總量不變；開放容器看有無氣體逸出

Q7Q8Q10Q21Q22Q24Q26Q28
兩種情境解法
  • 密閉
    反應前總質量 = 反應後總質量，直接加減求未知 Q8：前 6+58+5+23=92g，後 X+10+32+0=92g → X=50g
  • 開放
    逸散氣體質量 = 反應前容器質量 − 反應後容器質量 Q10：360g − 344g = 16g O₂ 逸散
比例計算（係數比→質量比）
  • 1
    係數 × 分子量 = 該物質的「質量基準」 CaCO₃→CO₂：係數都是1，質量比 = 100 : 44
  • 2
    列比例：（已知質量 / 已知基準）= （未知質量 / 未知基準） Q7：x/100 = 176/44 → x = 400g
  • 3
    題組題：先判斷哪個先用完（限量試劑），再算剩餘和產物 Q21：SiCl₄=2mol，H₂=6mol，反應需H₂=4mol → SiCl₄先用完
圖形判斷題（Q24 / Q26）
質量-時間圖下降 → 有氣體逸出（開放容器）。只生成沉澱的反應不會讓質量下降。
加熱前-後質量圖斜率 → 斜率 = (原料+加入物)/原料，從斜率可以反推質量比。
補充變化題 — 練習看看
在開放容器中，10g 碳酸鈣加入足量鹽酸完全反應後，容器內質量變化如何？（CaCO₃ 分子量=100，CO₂=44） ▼
1
CaCO₃ + 2HCl → CaCl₂ + H₂O + CO₂↑
2
10g CaCO₃ = 0.1 mol → 生成 0.1 mol CO₂ = 4.4g 逸散
3
容器內質量比反應前減少 4.4g（CO₂跑掉了）
密閉容器中：甲 + 乙 → 丙 + 丁。反應前甲=10g、乙=20g，反應後甲=4g、丙=18g，求乙和丁各剩多少？ ▼
1
反應前總質量 = 10 + 20 = 30g
2
甲消耗 = 10−4 = 6g；丙生成 18g
3
乙消耗 + 甲消耗 = 丙 + 丁 → 6 + 乙消耗 = 18 + 丁
又乙消耗 + 丁 + 4 + 18 = 30 → 乙消耗 = 8，所以乙剩 12g、丁 = 14g
已知銅 + 氧 → 氧化銅，質量比 Cu:O = 4:1。若有 20g 銅完全氧化，生成氧化銅幾克？消耗氧幾克？ ▼
1
Cu : O = 4 : 1 → 20g Cu 消耗 O = 20 × (1/4) = 5g
2
生成 CuO = 20 + 5 = 25g（質量守恆）
已知甲乙兩金屬各與鹽酸反應，甲 12g 產生 1g H₂，乙 23g 也產生 1g H₂。兩個反應的鹽酸用量是否相同？ ▼
兩者都產生 1g H₂ = 0.5 mol H₂，依 M + 2HCl → MCl₂ + H₂，都需要 1 mol HCl = 36.5g。

→ 鹽酸用量相同（與金屬種類和質量無關，只和產生的H₂量有關）
→ 但產物總質量不同（甲的氯化物 vs 乙的氯化物分子量不同）
04

莫耳比例計算

係數比 = 莫耳比，是整章最核心的概念

Q9Q11Q12Q13Q14Q25Q27Q29Q30
三個核心公式
  • ①
    莫耳數 = 質量 ÷ 分子量 28g N₂ ÷ 28 = 1 mol；32g O₂ ÷ 32 = 1 mol
  • ②
    係數比 = 莫耳數比（同溫同壓下也等於體積比） 但不等於質量比！質量比要再乘各自的分子量
  • ③
    限量試劑：哪個先用完，哪個決定產物量 比較「有多少」和「需要多少」，較不夠的那個是限量試劑
解題流程（通用步驟）
  • 1
    寫出平衡方程式，確認係數
  • 2
    把所有質量換成莫耳數（÷分子量）
  • 3
    找限量試劑（若題目有給多種反應物）
  • 4
    用莫耳比算出產物或剩餘物的莫耳數
  • 5
    若要求質量，再乘回分子量
特殊考法整理
考法題號解題關鍵
「哪個比值不等於係數比」 Q9 質量比 ≠ 係數比，要乘分子量
密閉容器氣體密度 Q11 質量守恆→密度 = 總質量 ÷ 體積（不用算每種物質）
從質量守恆反推分子量 Q12 係數×分子量加總兩邊相等：3×20 + B = 2×40
推算金屬元素 Q27 列方程式求 M，代入原子量表對照
燃燒耗O₂比較 Q29 分子本身含 O 的，對外需求的 O₂ 較少
實驗數據找限量試劑 Q30 觀察何時產物不再隨原料增加而增加
補充變化題 — 練習看看
N₂ + 3H₂ → 2NH₃。取 14g N₂ 與 6g H₂ 反應，哪個是限量試劑？可生成多少 NH₃？ ▼
1
N₂ = 14/28 = 0.5 mol；H₂ = 6/2 = 3 mol
2
0.5 mol N₂ 需要 1.5 mol H₂，有 3 mol H₂，N₂ 是限量試劑
3
0.5 mol N₂ → 1 mol NH₃ = 17g；H₂ 剩 3−1.5 = 1.5 mol = 3g
2SO₂ + O₂ → 2SO₃。取 4 mol SO₂ 與 1.5 mol O₂，反應後各有什麼、多少莫耳？ ▼
1
4 mol SO₂ 需要 2 mol O₂，但只有 1.5 mol → O₂ 是限量試劑
2
1.5 mol O₂ 消耗 SO₂ = 3 mol，生成 SO₃ = 3 mol
3
剩餘：SO₂ = 4−3 = 1 mol；O₂ = 0；SO₃ = 3 mol
密閉容器中 A + B → C，體積固定 10L，反應前質量共 60g，反應後測得密度為多少？（分子量 A=20, B=40, C=60） ▼
密閉容器 → 總質量守恆 = 60g，體積不變 = 10L

密度 = 60 ÷ 10 = 6 g/L（不需要知道反應了多少，直接算）
CH₄ 與 C₂H₆ 各燃燒 1 mol，哪個耗 O₂ 多？哪個生成水多？ ▼
CH₄ + 2O₂ → CO₂ + 2H₂O　→ 耗 O₂ = 2 mol，水 = 2 mol
C₂H₆ + 7/2 O₂ → 2CO₂ + 3H₂O　→ 耗 O₂ = 3.5 mol，水 = 3 mol

C₂H₆ 耗 O₂ 多、生成水多（碳數更多 + 氫原子更多）
已知 2XO + O₂ → 2XO₂，若 12g XO 完全反應後得 16g XO₂，求 X 的原子量 ▼
1
設 X 原子量 = M，XO 分子量 = M+16
2
莫耳數相同（係數2:2=1:1）：12/(M+16) = 16/(M+32)
3
12(M+32) = 16(M+16) → 12M+384 = 16M+256 → 4M = 128 → M = 32（硫 S）

手寫題型

計算題、方程式撰寫、進階題——常見考法整理

方程式撰寫題 — 高頻反應
反應方程式類型
鈉燃燒4Na + O₂ → 2Na₂O化合
SO₂溶於水SO₂ + H₂O → H₂SO₃化合
氧化汞分解2HgO → 2Hg + O₂↑分解
鈉與水2Na + 2H₂O → 2NaOH + H₂↑取代
AgNO₃ + NaClAgNO₃ + NaCl → AgCl↓ + NaNO₃複分解
易忘符號
有氣體生成要寫 ↑；有沉澱生成要寫 ↓（在水溶液中才需要標）
燃燒計算題 — 推分子式
  • 1
    燃燒產物中的 CO₂ → 數 C 的個數；H₂O → 數 H 的個數 每 mol CO₂ 含 1 mol C；每 mol H₂O 含 2 mol H
  • 2
    若是碳氫化合物（無O），分子式直接由C和H個數決定 每分子含4個C、8個H → C₄H₈
  • 3
    若可能含O，用「質量守恆」推算：原料質量 − C的質量 − H的質量 = O的質量
濃度計算題 — 重量百分濃度
  • 公式
    重量百分濃度 = 溶質質量 ÷ 溶液總質量 × 100% 溶液總質量 = 體積 × 密度（g = mL × g/cm³）
  • 注意
    混合多組溶質時，先把各組換算成同單位（克）再加總 克數直接加；莫耳數要乘分子量換算成克
半衰期計算題（段考第六題類型）
  • 1
    每過一個半衰期，濃度減半 初始濃度 × (1/2)ⁿ，n = 經過的半衰期次數
  • 2
    「高於 X 呈陽性」→ 濃度剛好等於 X 時，已不呈陽性，可以參加 段考：48h 時濃度 = 1.0 μg/mL，不高於 1.0，所以 48h 後可以參加
補充變化題 — 手寫練習
某碳氫化合物 0.1 mol 完全燃燒，得 8.8g CO₂ 和 9g H₂O，求分子式並寫燃燒反應式 ▼
1
CO₂ = 8.8/44 = 0.2 mol → 每分子含 0.2/0.1 = 2 個 C
2
H₂O = 9/18 = 0.5 mol → H = 0.5×2/0.1 = 10 個 H
3
分子式 C₂H₁₀？ → 不合理（氫太多），重算：每分子H = 1.0/0.1 = 10，分子式 C₂H₁₀ 不存在
實際：C₂H₅OH（乙醇）也含O，但題目說碳氫化合物，C₂H₁₀ 不可能存在，需重新確認題目數據
4
若 H₂O = 0.5 mol、每分子 H = 5：C₂H₆（乙烷），燃燒式：2C₂H₆ + 7O₂ → 4CO₂ + 6H₂O
某藥物半衰期為 8 小時，初始血液濃度 32 μg/mL，問多少小時後濃度低於 1 μg/mL？ ▼
1
每8h減半：32 → 16 → 8 → 4 → 2 → 1
2
共5次半衰期 = 40小時後，濃度 = 1 μg/mL
3
題目問「低於 1」→ 需要再過一次半衰期，即 48小時後濃度 = 0.5 μg/mL
寫出「鐵與稀硫酸反應」的化學方程式，並指出反應類型 ▼
Fe + H₂SO₄ → FeSO₄ + H₂↑

類型：取代反應（鐵把氫踢走，佔據硫酸中的位置）
注意：係數全為1（已平衡），H₂ 要寫 ↑
平衡 Al + Fe₂O₃ → Al₂O₃ + Fe（鋁熱反應），並寫出反應類型 ▼
1
設 Al₂O₃ 係數 = 1 → Al = 2，O = 3
2
Fe₂O₃ 提供 3 個 O → 係數 = 1；Fe = 2
3
2Al + Fe₂O₃ → Al₂O₃ + 2Fe，類型：取代反應（Al 活性 > Fe，把 Fe 踢走）

Q1
平衡係數的目的是符合哪個定律？
定比 / 倍比 / 原子說 / 分子說
(C) 原子說
▾
達爾頓的「原子說」：原子不會憑空出現，也不會憑空消失。
平衡係數讓原子個數守恆，符合原子說。
選項定義是否符合
定比定律同一化合物各元素質量比固定✗ 描述化合物性質
倍比定律兩種元素形成多種化合物，質量比為簡單整數比✗ 描述多種化合物
原子說原子不滅 → 方程式兩邊原子總數相等✓ 正確
分子說描述物質由分子組成✗ 不描述反應
Q2
化學方程式撰寫原則，甲乙丙丁哪些正確？
甲：用「=」連接 / 乙：條件標箭頭上方 / 丙：係數非最小代表反應不存在 / 丁：分子量與係數無關
(D) 乙、丁

三-1
某碳氫化合物0.2mol完全燃燒，產生35.2g CO₂及0.8mol H₂O，求分子式和耗O₂量
共4分
C₄H₈，耗O₂ 38.4g
▾
1
求CO₂莫耳數：35.2 ÷ 44 = 0.8mol CO₂
每mol化合物產生 0.8÷0.2 = 4mol CO₂ → 含 4個C
2
求H個數：0.8mol H₂O / 0.2mol = 4mol H₂O per mol → 每個分子含 8個H
3
分子式：C₄H₈（無O，純碳氫化合物）
4
燃燒耗O₂：C₄H₈ + 6O₂ → 4CO₂ + 4H₂O
0.2mol × 6 = 1.2mol O₂ = 38.4g
三-2
甲24.5g + 乙0.3mol + 丙4.9g 硫酸，配成500mL、密度1.2g/cm³溶液，求重量百分濃度
共4分
9.8%
▾
1
各組H₂SO₄質量：
甲：24.5g（直接給）
乙：0.3mol × 98g/mol = 29.4g
丙：4.9g（直接給）
2
H₂SO₄總質量：24.5 + 29.4 + 4.9 = 58.8g
3
溶液總質量：500mL × 1.2g/mL = 600g
4
重量百分濃度：(58.8 ÷ 600) × 100% = 9.8%
四
K₂FeO₄ + H₂O → O₂ + Fe₂O₃ + KOH，以最簡整數平衡
共3分
4:10:3:2:8
▾
4K₂FeO₄ + 10H₂O → 3O₂ + 2Fe₂O₃ + 8KOH
1
先平衡Fe：設Fe₂O₃係數=1 → 需要2個K₂FeO₄
2
平衡K：2個K₂FeO₄ → K=4 → KOH係數=4
3
平衡H：4個KOH → H=4 → H₂O係數=2
4
平衡O後出現分數，全體×2 → 係數4:10:3:2:8
六
安非他命半衰期12小時，誤服56mg，血液3.5L，濃度何時降到1.0μg/mL以下？
共3分
48小時後
▾
1
轉換單位：56mg = 56,000μg；3.5L = 3,500mL
初始濃度 = 56,000 ÷ 3,500 = 16 μg/mL
2
每12小時減半：
0h: 16 → 12h: 8 → 24h: 4 → 36h: 2 → 48h: 1 μg/mL
3
48h 時濃度剛好 = 1.0（題目說「高於1.0」才呈陽性）
→ 最早在48小時後可以參加比賽
