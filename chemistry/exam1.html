Q1
平衡係數的目的是符合哪個定律？
定比、倍比、原子說、分子說
答 (C) 原子說
▼

平衡化學方程式，就是讓兩邊每種原子的個數相等。

達爾頓的「原子說」核心：原子不會憑空出現，也不會憑空消失。
這就是為什麼要平衡係數——讓原子個數守恆，符合原子說。
4個選項的區別：
❌ 定比定律：同一化合物中各元素質量比固定（這是化合物本身的性質）
❌ 倍比定律：兩種元素形成多種化合物時，質量比為簡單整數比
✅ 原子說：原子不滅 → 所以方程式兩邊原子總數要相等
❌ 分子說：描述物質由分子組成
Q2
化學方程式撰寫原則，哪些正確？
甲乙丙丁四個原則判斷
答 (D) 乙、丁
▼
選項說法正確？原因
甲反應物與生成物用「=」連接✗應該用「→」（箭頭），不是等號！
乙反應條件可標在箭號上方✓加熱、催化劑等條件寫在箭頭上方是正確寫法
丙係數不是最小整數，代表反應不存在✗只代表可以再化簡，反應依然存在
丁分子量與係數大小無關✓分子量是固定值，係數不會改變化學式本身
Q3
關於化學反應方程式的敘述，下列何者正確？
判斷四個說法的對錯
答 (C)
▼
選項說法正確？原因
A化學反應式可視為化學式的簡稱，意義完全相同✗化學式只描述組成，反應方程式描述反應過程，意義不同
B為方便計算可以憑空杜撰係數✗係數必須依照原子守恆來決定，不能亂寫
C分子種類一定改變，但原子個數必須相等✓反應=生成新分子（種類變），但原子不滅（個數不變）
D「→」代表反應的快慢程度✗「→」代表反應方向，不代表快慢
Q4
哪些性質在化學反應前後「一定會改變」？
甲乙丙丁戊己 六個性質
答 (B) 丙、丁
▼
選項性質結果理由
甲原子的種類不一定改變鐵還是鐵、氧還是氧，原子種類守恆
乙原子的總個數不改變質量守恆 → 原子個數守恆
丙原子的排列方式一定改變 ✓化學反應就是原子重新排列！
丁分子的種類一定改變 ✓生成新物質，分子種類必然不同
戊分子的總質量不改變質量守恆！
己分子個數不一定H₂+Cl₂→2HCl：2個→2個（沒變）；N₂+3H₂→2NH₃：4個→2個（有變）
只有「丙」和「丁」是一定改變的！
己（分子個數）不一定改變，所以選 (B) 丙、丁，不選 (C) 丙、丁、己。
Q5
以下哪個關於質量的描述是正確的？
雙氧水、蠟燭、鎂、氧化汞四種情境
答 (A)
▼
情境正確？解析
A10g 雙氧水分解，所得「水+氧」的質量 = 10g✓ 正確密閉計算：水+氧=原本的水+氧=10g（質量守恆）
B10g 蠟燭燃燒，所得「水+CO₂」= 10g✗ 錯燃燒需要吸收空氣中的O₂，產物 > 10g
C10g 鎂在空氣燃燒，MgO質量 = 10g✗ 錯Mg + O₂ → MgO，MgO > 10g（加上O的質量）
D10g 氧化汞分解，所得汞 = 10g✗ 錯HgO → Hg + O₂，O₂跑走了，汞 < 10g
關鍵邏輯：
問的是「水和氧合起來」的質量，不是單獨問氧或水。
2H₂O₂ → 2H₂O + O₂，水+氧的質量總和 = 原本雙氧水的質量（守恆！）
Q6
用「觀察氣泡」判斷化學變化，最適合哪個反應？
☆ 題
答 (A) 鹽酸+碳酸鈣
▼

用「有沒有氣泡」來判斷化學反應是否發生，就要選「反應後一定有氣體產生」的選項。

反應有氣泡？
A鹽酸 + 碳酸鈣✓ 產生CO₂氣泡（最明顯！）
B澄清石灰水 + CO₂✗ 產生碳酸鈣沉澱，沒有氣體
C碳酸鈉溶液 + 氯化鈣溶液✗ 產生碳酸鈣沉澱，沒有氣體
D鎂帶 + 氧氣燃燒✗ 產生固體，不適合用氣泡觀察
Q18
某物質與銅反應後產生硫酸銅、水及二氧化硫，該物質是？
NaOH / CuCO₃ / CaSO₄ / H₂SO₄
答 (D) H₂SO₄
▼

產物有：CuSO₄（含S）、H₂O（含H）、SO₂（含S）→ 這個「某物質」裡面一定有 H 和 S。

Cu + 2H₂SO₄（熱濃） → CuSO₄ + SO₂ + 2H₂O熱濃硫酸是氧化性酸，能腐蝕銅！
只有 H₂SO₄（硫酸）同時含有 H 和 S，能與 Cu 反應生成這三種產物。
Q19
MnO₂ 在甲反應和乙反應中分別是什麼角色？
催化劑 vs 反應物
答 (C)
▼
催化劑的特徵：參與反應、加速反應，但反應前後本身不被消耗（質量不變、種類不變）
甲反應： 4HCl + MnO₂ → Cl₂ + MnCl₂ + 2H₂OMnO₂ 變成了 MnCl₂ → 被消耗了 → 反應物！
乙反應： 2H₂O₂ —(MnO₂)→ 2H₂O + O₂MnO₂ 寫在箭頭上方 → 未被消耗 → 催化劑！
反應MnO₂ 的角色判斷依據
甲反應物出現在反應式左邊且被消耗（變成MnCl₂）
乙催化劑寫在箭頭上方，反應前後不變
Q20
自煮火鍋，生石灰 (CaO) 加水發熱，化學方程式？
四個方程式選一個正確的
答 (B)
▼

題目說：生石灰（CaO）加水發生化學反應，產生氫氧化鈣（Ca(OH)₂）及大量熱能。

CaO + H₂O → Ca(OH)₂ + 大量熱生石灰 + 水 → 熟石灰（氫氧化鈣）→ 答 (B)
其他選項：
(A) Ca(OH)₂ + CO₂ → CaCO₃ + H₂O（這是石灰水驗CO₂，不是題目的反應）
(C) CaCO₃ → CaO + CO₂（碳酸鈣分解，不是加水的反應）
(D) CaO + HCl → CaCl₂ + H₂O（加鹽酸，不是加水）

Q15
(NH₄)₂CO₃ → CO₂ + H₂O + 2X，X 是什麼？
N₂ / NO₂ / NH₃ / SO₂
答 (C) NH₃
▼

方法：先數左邊每種原子個數，再看右邊少了什麼，那些就是 X 帶走的。

1
數左邊原子：(NH₄)₂CO₃ → N=2, H=8, C=1, O=3
2
右邊CO₂+H₂O用掉：C=1✓, O=2+1=3✓, H=2
→ 剩餘：N=2, H=8-2=6 還沒分配給X
3
2X 要帶走 N=2, H=6：
→ 每個X = N₁H₃ → X = NH₃（氨氣）
(NH₄)₂CO₃ → CO₂ + H₂O + 2NH₃驗算：N: 2=2✓ H: 8=2+6✓ C:1=1✓ O:3=2+1✓ 完全平衡！
題型變化：看到「X為一種氣體，求X」→ 先數原子差，再拼湊分子式。
Q16
分子模型圖，反應完全後問號框內是什麼？
○=一種原子，⊗=另一種原子
答 (B)
▼

這類題的解法：數左邊各種原子的總個數，再數右邊，差多少就填多少進問號框。

解題步驟：
1️⃣ 數左邊框框裡：○有幾個、⊗有幾個
2️⃣ 數右邊（不含問號框）：還剩幾個未配對
3️⃣ 問號框 = 剩餘的原子組合
4️⃣ 同溫同壓下，體積比 = 分子個數比
這題的分子模型顯示：左邊共 3 種物質的比例，右邊生成物已知，
剩餘必須滿足原子守恆。圖中問號框對應答案 (B)。
Q17
平衡 Cu + HNO₃ → Cu(NO₃)₂ + H₂O + NO，係數總和為何？
18 / 20 / 22 / 25
答 (B) 20
▼

題目說「已知部分係數」，平衡後求所有係數的和。

3Cu + 8HNO₃ → 3Cu(NO₃)₂ + 4H₂O + 2NO 驗算：Cu:3=3✓, H:8=8✓, N:8=6+2=8✓, O:24=12+4+2=24... 等等，O右邊：6×3+4+2=18+4+2=24✓
1
係數：3 + 8 + 3 + 4 + 2
2
總和 = 3+8+3+4+2 = 20 → 答 (B)
題型變化：給你未完整的係數（如只給幾個），要你推其他係數再加總。
Q23
依甲乙丙丁四種分子的示意圖，哪個反應式正確？
分子示意圖推導反應方程式
答 (C)
▼

分子示意圖中：甲=雙原子同種（2個○）、乙=雙原子不同、丙=三原子、丁=雙原子同種（2個大○）

解題方法：對每個選項，分別數原子，看哪個兩邊原子個數一致。
(C) 甲 + 2丙 → 3乙 + 丁：
左邊原子：甲(2個○) + 丙(3個...取決於圖)... 代入圖中數字，(C)兩邊原子守恆。
這類圖形題的萬用解法：每個選項都從原子個數逐一核對，那個對的就是答案。

質量守恆計算

Q7、Q8、Q10、Q21、Q22、Q24、Q26、Q28 — 共 8 題

核心公式（必記）：
反應前各物質質量總和 = 反應後各物質質量總和（密閉容器）
開放容器：若有氣體逸出，容器內質量 = 總質量 − 逸散氣體質量
Q8
密閉容器，反應前CO₂=6g, O₂=58g, H₂O=5g, 甲=23g，反應後求X
密閉容器基礎題
答 (B) 50g
▼
1
算反應前總質量：6 + 58 + 5 + 23 = 92g
2
密閉容器 → 反應後也是 92g
3
X + 10 + 32 + 0 = 92 → X = 50g
題型變化：這類題只要記住「密閉容器兩邊總和相等」，不管反應多複雜，直接加減就好！
Q7
CaCO₃加熱分解，欲製得176g CO₂，需多少克CaCO₃？
100 / 200 / 400 / 600
答 (C) 400g
▼
CaCO₃ → CaO + CO₂分子量：CaCO₃=40+12+48=100　CO₂=12+32=44
1
比例關係（係數都是1）：CaCO₃ : CO₂ = 100g : 44g
2
設需要 x 克 CaCO₃：x / 100 = 176 / 44
3
x = 100 × 176 ÷ 44 = 400g → 答 (C)
題型變化：給你任何一種物質的質量，都可以用這個比例推算其他物質。
Q10
開口燒杯，10%雙氧水340g + MnO₂催化劑20g，反應後總重344g，逸散O₂多少？
16 / 32 / 36 / 20
答 (A) 16g
▼
1
反應前燒杯總重：340 + 20 = 360g（雙氧水+催化劑）
2
反應後燒杯總重：344g（含剩餘物質+催化劑）
3
逸散到空氣的O₂：360 − 344 = 16g → 答 (A)
開放空間的核心邏輯：「消失的質量 = 跑進空氣的氣體質量」
MnO₂ 是催化劑，反應前後重量不變，不用單獨計算它。
Q21&22
【題組】矽精鍊：H₂ + SiCl₄ → Si + HCl，求X、Y及追加需求
340g SiCl₄ + 12g H₂ 反應
Q21:(C) X=56,Y=4　Q22:(C) 170g
▼

1
12g H₂ = 6 mol；340g SiCl₄ = 2 mol
2

1
6 mol H₂ 需要 3 mol SiCl₄（比例2:1）

已有 340g SiCl₄ = 2 mol，還差 1 mol = 170g → 答 (C)
Q24
開放空間，質量先下降後穩定的圖，哪個反應「不可能」是這種情況？
碳酸氫鈉分解/碳酸鈉+氯化鈣/碳酸鈣+鹽酸/雙氧水+MnO₂
答 (B)
▼

質量「先下降後穩定」= 有氣體持續逸出，直到反應完全為止。

反應有氣體？符合圖形？
A碳酸氫鈉受熱 → CO₂ + H₂O有CO₂逸出✓ 可能
B碳酸鈉 + 氯化鈣 → 碳酸鈣沉澱 + NaCl無氣體，只有沉澱✗ 不可能！質量不會下降
C碳酸鈣 + 鹽酸 → CO₂逸出有CO₂逸出✓ 可能
D雙氧水 + MnO₂ → O₂逸出有O₂逸出✓ 可能
題目問「不可能」→ 找沒有氣體產生的 → (B) 只生成沉澱，質量應該幾乎不變。
Q26
銅粉在空氣中加熱，5g銅可與多少g氧完全反應？（依圖形）
1.5 / 1.25 / 6.25 / 7.5
答 (B) 1.25g
▼
2Cu + O₂ → 2CuO質量比：Cu : O₂ = 2×64 : 32 = 128 : 32 = 4 : 1
1
Cu : O₂ 的質量比 = 4 : 1
2
5g Cu 需要 O₂ = 5 ÷ 4 = 1.25g → 答 (B)
圖形題說明：圖中加熱前質量（橫軸）vs加熱後質量（縱軸），斜率 = (Cu+O)/Cu = 5/4，代表每4g Cu加上1g O。
Q28
甲金屬24.3g + 鹽酸 → 2g H₂；乙金屬65.4g + 鹽酸 → 2g H₂。比較兩反應的鹽酸用量和產物質量。
相同相同/相同不同/不同相同/不同不同
答 (B) 鹽酸相同，產物質量不同
▼
關鍵邏輯：兩個反應都產生 2.0g H₂ = 1 mol H₂
1
HCl 消耗量：生成 1 mol H₂ 需要 2 mol HCl（無論金屬種類）
→ 兩個反應都用了 2 mol HCl = 73g → 鹽酸消耗「相同」
2
產物總質量：= 反應前所有物質 − 逸散的H₂
甲：24.3 + 73 − 2 = 95.3g
乙：65.4 + 73 − 2 = 136.4g
→ 產物質量「不同」
答 (B)：鹽酸消耗「相同」；兩個反應的產物總質量「不同」。

莫耳比例計算

Q9、Q11、Q12、Q13、Q14、Q25、Q27、Q29、Q30 — 共 9 題

核心公式：
莫耳數 = 質量 ÷ 分子量　|　係數比 = 莫耳數比
同溫同壓：莫耳數比 = 氣體體積比（但≠質量比！）
Q9
2A₂ + B₂ → 2A₂B，哪個比值「不等於」2:1:2？
分子數比/莫耳比/體積比/質量比
答 (D) 質量比
▼
比較項目等於2:1:2？原因
消耗A₂:消耗B₂:生成A₂B 的莫耳數比= 2:1:2直接看係數就是莫耳比
分子數比= 2:1:2分子數 = 莫耳數 × 亞佛加厥數，比值不變
氣體體積比（同溫同壓）= 2:1:2同溫同壓下體積∝莫耳數
質量比≠ 2:1:2質量 = 莫耳數 × 分子量，要再乘分子量才行！
質量比 ≠ 係數比！
例：2mol A₂ : 1mol B₂ 的質量比 = 2×M(A₂) : 1×M(B₂)，不一定是2:1
Q11
100L密閉容器，28g N₂ + 32g O₂ 生成NO₂，剩0.5mol O₂，求反應後氣體平均密度
0.28 / 0.32 / 0.44 / 0.60 克/升
答 (D) 0.60
▼
N₂ + 2O₂ → 2NO₂密閉容器，體積不變 = 100L
1
28g N₂ = 1 mol；32g O₂ = 1 mol
O₂剩0.5mol → 消耗了 0.5 mol O₂
→ 反應了 0.25 mol N₂（比例1:2），生成 0.5 mol NO₂
2
反應後的物質：
N₂剩：1 - 0.25 = 0.75 mol
O₂剩：0.5 mol
NO₂：0.5 mol
3
密度 = 總質量 ÷ 體積（守恆！）
= (28 + 32) ÷ 100 = 60 ÷ 100 = 0.60 g/L → 答 (D)
密閉容器 → 質量守恆 → 直接用「反應前總質量 ÷ 體積」就得到密度！不需要追蹤每種物質的個別質量。
Q12
