7-1 · 質量守恆定律
物質不會憑空消失，也不會無中生有
法國科學家拉瓦節在 18 世紀末，用精密的天平反覆實驗，確立了化學史上最重要的定律之一。

實驗 7-1：碳酸鈉 + 氯化鈣

碳酸鈉溶液與氯化鈣溶液裝在密封裝置兩側，測量實驗前總質量 M₁ 克。倒置使兩液混合，反應後測量總質量 M₂ 克。

Na₂CO₃(aq) + CaCl₂(aq) → CaCO₃↓（白色）+ 2NaCl(aq)

結果：M₁ ＝ M₂ ——反應前後總質量相等

質量守恆定律

在密閉容器中進行反應時，反應前後系統的總質量不變。
對一般化學反應：反應物消耗的總質量 ＝ 生成物生成的總質量

驗證質量守恆定律，必須在密閉系統中進行，否則氣體散失或被吸收，質量看起來會改變，無法正確驗證。
延伸知識 — 拉瓦節與斷頭台

拉瓦節（Lavoisier，1743–1794）出身貴族，靠精密的定量實驗革命了化學。他發現了氧氣的角色、推翻燃素說、確立質量守恆。

諷刺的是，他在法國大革命中以「收稅欺詐罪」被送上斷頭台。數學家拉格朗日說：「砍下這顆頭只要一瞬間，但要再生出這樣的一顆頭，恐怕需要一百年。」

延伸知識 — 質量守恆在宇宙中的極限

普通化學反應中，質量守恆完全成立。但在核反應（核分裂、核融合）中，愛因斯坦的 E=mc² 告訴我們：極微量的質量可以轉換成巨大能量。太陽每秒把約 400 萬噸的質量轉換成光和熱！

不過，化學反應中的質量轉換太微小，儀器根本測不出來，所以對化學來說，質量守恆是完全正確的定律。

這一課

  • Na₂CO₃ + CaCl₂ → CaCO₃↓（白色沉澱）+ 2NaCl，密閉容器 M₁ = M₂
  • 反應物消耗的總質量 = 生成物生成的總質量
  • 驗證時必須用密閉系統（防止氣體逸散或被吸收）
7-1 · 質量守恆的應用
看起來消失了？看起來變重了？都是假象
質量守恆從來不會破壞——只是在開放系統中，氣體可能散失或被吸收，讓我們誤以為質量變了。
例一 — 蠟燭燃燒
現象：蠟燭越燒越短，質量好像減少
真相：蠟燭 + O₂ → CO₂ + H₂O。產生的 CO₂ 和水蒸氣散失到空氣中。
在密閉容器中
把消耗的 O₂ 和產生的 CO₂、H₂O 都算入系統，總質量完全不變。

反應消耗的蠟燭 + O₂ ＝ 生成的CO₂ + H₂O
若 x 克蠟燭燃燒後剩 w 克，生成 y 克水和 z 克 CO₂：
O₂ 消耗量 ＝ y + z − (x − w) ＝ y + z + w − x 克

例二 — 鐵生鏽
現象：鐵生鏽後質量好像增加
真相：鐵生鏽時吸收了空氣中的 O₂ 和水蒸氣，所以鐵鏽比鐵重。
在密閉容器中
把鐵吸收的 O₂ 和 H₂O 都計入系統，總質量仍然不變。

質量守恆計算器

反應情境 原來總質量（克） 逸出氣體質量（克）
延伸知識 — 為什麼鑽石會燒掉，黃金卻不會？

鑽石是純碳（C），會和氧氣發生化學反應（燃燒）：C + O₂ → CO₂，產生的 CO₂ 氣體散失，所以鑽石在高溫下真的會「燃燒消失」。

黃金（Au）化性非常穩定，在一般溫度下不與 O₂ 反應，所以不會生鏽也不會燃燒，這就是它能保值幾千年的原因。

這一課

  • 蠟燭越燒越短：CO₂ 和 H₂O 散失到空氣中（非質量真的消失）
  • 鐵生鏽後較重：吸收了 O₂ 和水蒸氣
  • 開口容器剩餘質量 = 原總質量 − 逸出氣體質量
  • 密閉容器：反應前後總質量永遠不變
7-1 · 開放 vs 密閉
容器有沒有蓋子，計算方式完全不同
情境開放容器密閉容器
大理石 + 鹽酸質量下降（CO₂ 逸出）質量不變
蠟燭燃燒質量下降（CO₂、H₂O 逸出）質量不變
鐵生鏽質量上升（吸收 O₂、H₂O）質量不變
雙氧水分解質量下降（O₂ 逸出）質量不變

計算例題

例1：密閉容器密度
100 升真空密閉容器通入 28g N₂ 和 32g O₂，加熱生成 NO 氣體，反應後容器體積和溫度不變，氣體平均密度為？
解：密閉容器總質量不變 = 28+32 = 60 克，密度 = 60÷100 = 0.6 g/L
例2：開口燒杯計算
20% 雙氧水 170g + MnO₂ 10g，完全反應後產生 16g O₂ 逸出，燒杯中剩下物質共重？
解：(170+10) − 16 = 164 克（MnO₂ 是催化劑，質量不變，仍在燒杯中）
例3：蠟燭燃燒計算
若 x 克蠟燭燃燒，生成 y 克水與 z 克 CO₂，燃燒後尚餘蠟燭 w 克，則消耗了多少克 O₂？
解：(x−w) + O₂ = y+z，O₂ = y+z−(x−w) = y+z+w−x 克
例4：A+B→C（質量守恆）
反應前 A 有 6 克，B 有 8 克，反應後 B 耗盡，A 剩 2 克，則 C 為多少克？
解：A 消耗 = 6−2 = 4 克，C = 4+8 = 12 克

這一課

  • 開放容器：產生氣體 → 質量減少；吸收氣體 → 質量增加
  • 密閉容器：質量永遠不變
  • 計算公式：剩餘質量 = 原總質量 − 逸出氣體質量

7-2 · 化學反應方程式
用化學語言描述反應過程
化學反應方程式（簡稱反應式）用化學式、數字、符號，把一個化學反應完整表達出來，必須根據實驗結果而定。

基本格式

反應物₁ + 反應物₂ → 生成物₁ + 生成物₂

四個寫法步驟

① 用化學式表示反應物和生成物
多種反應物或生成物之間用「+」相連，反應物和生成物之間用「→」分隔。
② 標注反應條件
加熱（△）、加壓、催化劑名稱、電解（ε）等，標在箭號上方或下方。
③ 標注氣體和沉澱
生成物為氣體 → 右側標「↑」；生成物為沉澱 → 右側標「↓」。
④ 均衡（最重要！）
加上平衡係數，使兩邊每種原子數目相等，係數以最簡整數表示。

反應條件符號對照

符號意義例
△（箭號上方）加熱2HgO →△→ 2Hg + O₂
ε（箭號上方）電解（通電）2H₂O →ε→ 2H₂ + O₂
催化劑名稱加催化劑2H₂O₂ →MnO₂→ 2H₂O + O₂
↑氣態生成物CaCO₃+2HCl → CO₂↑+…
↓沉澱生成物…→ CaCO₃↓ + 2NaCl
延伸知識 — 為什麼方程式一定要均衡？

化學反應是原子重新排列的過程，道耳頓原子說告訴我們：原子既不會消失也不會無中生有。所以方程式兩邊的每一種原子個數必須完全相等——這就是質量守恆定律在化學式上的體現。

這一課

  • 反應式 = 化學式 + 係數 + 符號，根據實驗定
  • 條件：△加熱，ε電解，催化劑名稱標箭號上下
  • 氣態生成物↑，沉澱↓
  • 最後必須均衡（兩邊原子數相等，最簡整數）
7-2 · 均衡化學方程式
讓箭號兩邊的每種原子數目相等

均衡原則：箭號前後每種原子的數目相等，係數以最簡整數表示

均衡三步驟

步驟 a　先平衡「左右各只出現一次的原子」
找出只在反應物或只在生成物中出現一次的原子，先平衡它，避免改動係數時牽連太多。
步驟 b　最後再全部檢查一遍
確認箭號前後每一種原子的個數都完全相等。
步驟 c　係數化為最簡整數
若所有係數有公因數，需除以公因數（例如 2:4:4 → 化簡為 1:2:2）。

示範：H₂ + O₂ → H₂O

未均衡：H₂ + O₂ → H₂O　（O 左邊 2 個，右邊 1 個，不平衡）
補 O：H₂ + O₂ → 2H₂O　（O 平衡了，但 H 左 2 個，右 4 個）
補 H：2H₂ + O₂ → 2H₂O　✓ （H: 4=4，O: 2=2）
均衡結果：2H₂ + O₂ → 2H₂O

  • 先平衡左右各只出現一次的原子

延伸知識 — 煙火的顏色是化學反應

煙火不同的顏色來自不同金屬鹽類燃燒時釋放的特定光：鈉（Na）→橙黃色，鉀（K）→紫色，鋇（Ba）→綠色，鍶（Sr）→紅色，銅（Cu）→藍綠色。這就是「焰色反應」——化合反應的同時也是鑑定元素的方法。

延伸知識 — 汽車安全氣囊的秘密

汽車碰撞時，氣囊在 0.03 秒內充氣保護你。觸發的是一個分解反應：

2NaN₃ → 2Na + 3N₂↑

一個氣囊約裝 65g NaN₃（疊氮化鈉），分解後快速產生大量 N₂ 氣體充滿氣囊。反應後的金屬鈉再與其他物質反應，確保不留有毒物質。

延伸知識 — 鋁熱反應：焊接鐵軌的高溫反應

鐵路工人用「鋁熱反應」來焊接鐵軌：

Fe₂O₃ + 2Al → Al₂O₃ + 2Fe + 大量熱

這個取代反應的溫度可達 2500°C，產生的液態鐵直接填入缺口冷卻凝固——不需要電焊，只要化學反應就夠了。

這一課

  • 化合：A+B→C（多→一）
  • 分解：A→B+C（一→多）
  • 取代：AB+C→AC+B（單質取代化合物中的元素）
  • 複分解：AB+CD→AD+CB（互換成分，像互送禮物）

7-3 · 化學計量
方程式係數比 = 莫耳數比
學會了方程式，下一步是用它來計算「需要多少？能生成多少？」——這就是化學計量。

反應式中的平衡係數比 = 莫耳數比 = 分子數比 = 體積比（僅限氣體）

質量比 ≠ 係數比！質量比 = 係數比 × 各自分子量。例如 2H₂ + O₂ → 2H₂O 中，質量比 = 4:32:36 = 1:8:9，而非 2:1:2。

2H₂ + O₂ → 2H₂O 各種比值

比值H₂O₂H₂O說明
係數比212—
莫耳數比212= 係數比
分子數比212= 係數比
體積比212= 係數比（氣體才成立）
質量比43236= 1:8:9（≠ 係數比！）

三步驟換算程序

步驟① 將已知物質換算成莫耳數
克數 ÷ 分子量 = 莫耳數
步驟② 用平衡係數比算出目標物質的莫耳數
目標莫耳數 = 已知莫耳數 × （目標係數 ÷ 已知係數）
步驟③ 換算成所需單位
克數 = 莫耳數 × 分子量　　個數 = 莫耳數 × 6×10²³
克 ÷分子量 → 莫耳 ×係數比 → 目標莫耳 ×分子量 → 克
克 ← ×分子量 莫耳 ×(6×10²³) → 個數

限制試劑

當兩種反應物都有限量時，先耗盡的那一種叫限制試劑，它決定了產物的最大量。

判斷限制試劑的方法

例：2Mg + O₂ → 2MgO，有 12g Mg 和 6g O₂：
12g Mg ÷ 24 = 0.5 mol Mg，需要 O₂ = 0.25 mol = 8g
但只有 6g O₂ = 0.1875 mol → O₂ 先耗盡 → O₂ 是限制試劑
MgO 最多 = 0.375 mol × 40 = 15 克

這一課

  • 平衡係數比 = 莫耳數比 = 分子數比 = 體積比（氣體）
  • 質量比 ≠ 係數比（要乘以各自分子量）
  • 三步驟：已知量→莫耳→係數比→目標莫耳→所求量
  • 限制試劑：先耗盡的反應物決定產物最大量
7-3 · 計算例題
逐步解析 8 道典型題目
例1：1 mol 鎂可與幾 mol 氧反應？（2Mg+O₂→2MgO）
Mg:O₂ = 2:1，1 mol Mg × (1/2) = 0.5 mol O₂
例2：24g 鎂和足量氧反應，生成幾克 MgO？（Mg=24）
24÷24=1 mol Mg → 1 mol MgO；MgO分子量=40，生成 40 克 MgO
例3：5 mol 鎂與多少克氧恰好完全反應而生成 200g MgO？
2Mg:O₂=2:1，5 mol Mg → 2.5 mol O₂ = 2.5×32 = 80 克 O₂
MgO = 5 mol × 40 = 200 克 ✓
例4：碘化鉀83g與足量硝酸鉛作用，生成碘化鉛沉澱幾克？（K=39,I=127,Pb=207）
2KI + Pb(NO₃)₂ → PbI₂↓ + 2KNO₃
KI分子量=166，83g÷166=0.5 mol KI → 0.25 mol PbI₂
PbI₂分子量=461，0.25×461 = 115.25 克
例5：使 0.1 mol KI 完全反應，至少需加入 0.5M 硝酸鉛水溶液幾 mL？
2KI:Pb(NO₃)₂=2:1，0.1 mol KI → 0.05 mol Pb(NO₃)₂
V = 0.05÷0.5 = 0.1 L = 100 mL
例6：40%、170g 雙氧水完全分解，可得 (1) 幾 mol O₂？(2) 幾克 O₂？
2H₂O₂ → 2H₂O + O₂；H₂O₂ = 170×40%=68g，68÷34=2 mol H₂O₂
(1) 1 mol O₂　(2) 1×32 = 32 克
例7：26g A 加熱完全分解→11g B + 氣體 C；C 與 X 克物質化合生成 21g D，則 X = ？
C = 26−11 = 15g；X = 21−15 = 6 克
例8：N₂+3H₂→2NH₃，氮氣和氫氣各15L（同溫同壓），最多得幾升氨？
H₂需要 N₂ 的 3 倍，但 15L N₂ 對 15L H₂ → H₂ 先耗盡（限制試劑）
15L H₂ × (2/3) = 10 升 NH₃，N₂ 剩 15−5 = 10L

7-3 · 莫耳濃度
每升溶液中有幾莫耳溶質
莫耳濃度（M）是化學中最常用的濃度單位，連接了「莫耳計算」和「溶液實驗」。

莫耳濃度（M）= 溶質莫耳數 ÷ 溶液體積（升）
M = n / V　單位：mol/L

莫耳濃度計算例題

例1：2 mol NaCl 配成 500 mL 溶液，M = ？
M = 2 ÷ 0.5 = 4 M
例2：14.8g Ca(OH)₂ 配成 10L，M = ？　（Ca=40）
Ca(OH)₂ 分子量 = 74，n = 14.8÷74 = 0.2 mol；M = 0.2÷10 = 0.02 M
例3：3M H₂SO₄ 150 mL 中含幾克硫酸？（S=32）
n = 3×0.15 = 0.45 mol；克數 = 0.45×98 = 44.1 克
例4：3M 150mL 硫酸操作後
(1) 倒掉 50mL → 濃度仍 3 M（倒掉溶液不改變濃度！）
(2) 加水稀釋至 180mL → M = 0.45÷0.18 = 2.5 M
(3) 蒸發至 120mL → M = 0.45÷0.12 = 3.75 M
例5：3M×150mL + 1M×250mL 混合後 M = ？
n總 = 3×0.15+1×0.25 = 0.70 mol；V總 = 400mL
M混 = 0.70÷0.40 = 1.75 M
倒掉溶液 → 濃度不變。加水才改變濃度（稀釋公式：M₁V₁ = M₂V₂）。

延伸知識 — 生理食鹽水的莫耳濃度

打點滴用的生理食鹽水是 0.9% NaCl 溶液（每 100mL 含 0.9g NaCl，分子量 58.5）。換算成莫耳濃度：M = 9g ÷ 58.5 ÷ 1L ≈ 0.154 M。

這個濃度跟人體血液的滲透壓非常接近，所以打進血管後細胞不會膨脹（水進入）也不會萎縮（水流出）。

延伸知識 — 酒測標準也是用濃度表示

台灣規定汽車駕駛人呼氣中酒精濃度不得超過 0.15 mg/L（即每升呼氣含 0.15 毫克酒精），以血液計算約為 0.03%。雖然不是用莫耳濃度，但背後的概念完全相同——用濃度來表示「每單位體積有多少溶質」。

這一課

  • M = n/V（升）；n = M×V；M = g/(MW×V)
  • 倒掉溶液不改變濃度；加水→濃度降；蒸發→濃度升
  • 稀釋公式：M₁V₁ = M₂V₂（溶質莫耳數不變）
  • 混合：M混 = (M₁V₁+M₂V₂)/(V₁+V₂)
7-3 · 補充★
鹼土族與鹵素的化學性質
不同的金屬元素對試劑的反應不同——利用這個差異，我們可以「鑑定」溶液中含有哪種元素。
鹼土族 — 加入 Na₂CO₃(aq) 的反應
化合物化學式含元素加入 Na₂CO₃ 後
氯化鈣CaCl₂Ca、Cl白色 CaCO₃ 沉澱
硝酸鈣Ca(NO₃)₂Ca、N、O白色 CaCO₃ 沉澱
氯化鎂MgCl₂Mg、Cl白色 MgCO₃ 沉澱
硝酸鎂Mg(NO₃)₂Mg、N、O白色 MgCO₃ 沉澱
氯化鉀KClK、Cl仍為澄清無色溶液
硝酸鉀KNO₃K、N、O仍為澄清無色溶液

鈣（Ca）、鎂（Mg）的化合物與 Na₂CO₃ 反應均會沉澱。鈉（Na）、鉀（K）的化合物則不會。
Ca、Mg 化性相似，在週期表同一直行，叫做鹼土族元素。

鹵素 — 加入 AgNO₃(aq) 的反應
化合物化學式含元素加入 AgNO₃ 後
氯化鈉 / 氯化鉀NaCl / KClCl白色 AgCl↓ 沉澱
溴化鈉 / 溴化鉀NaBr / KBrBr淡黃色 AgBr↓ 沉澱
碘化鈉 / 碘化鉀NaI / KII黃色 AgI↓ 沉澱

氯（Cl）、溴（Br）、碘（I）的化合物與 AgNO₃ 反應均會沉澱：
AgCl = 白色、AgBr = 淡黃色、AgI = 黃色。
Cl、Br、I 化性相似，在週期表同一直行，叫做鹵素元素。

鹵素反應方程式（複分解）

NaCl + AgNO₃ → NaNO₃ + AgCl↓（白）
NaBr + AgNO₃ → NaNO₃ + AgBr↓（淡黃）
NaI + AgNO₃ → NaNO₃ + AgI↓（黃）
延伸知識 — 這些性質怎麼應用在生活中？

飲用水消毒：自來水廠加氯氣（Cl₂）殺菌，水中就含 Cl⁻。用 AgNO₃ 一測，出現白色沉澱，就能確認有氯離子。

拍立得照片：傳統黑白照片用 AgBr 感光——光照使 AgBr 分解產生銀，不照光的地方保持透明。這就是鹵化銀對光敏感的原理。

這一課

  • 鹼土族（Ca、Mg）+ Na₂CO₃ → 白色沉澱；Na、K 不沉澱
  • 鹵素（Cl Br I）+ AgNO₃ → AgCl白 / AgBr淡黃 / AgI黃
  • 這些都是複分解反應（AB+CD→AD+CB）

7-1 質量守恆定律

在密閉容器中，反應前後系統的總質量不變。
反應物消耗的總質量 = 生成物生成的總質量

密閉容器 → 質量永遠不變
開放容器 → 產生氣體：質量減；吸收氣體：質量增
蠟燭越燒越短 → CO₂ 和 H₂O 散失（非質量消失）
鐵生鏽後較重 → 吸收了 O₂ 和水蒸氣
