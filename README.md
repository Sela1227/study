## 如何上傳到 GitHub Pages

1. github.com 建立帳號，新建 repository（名稱如 study，選 Public）
2. 點「uploading an existing file」，把 zip 解壓縮後把所有檔案拖進去
3. Commit changes
4. Settings → Pages → Branch:main / 資料夾:/(root) → Save
5. 約1分鐘後：https://你的用戶名.github.io/study/

## 版本命名

- 主版本（V1→V2）：大幅架構改動或設計重製
- 次版本（V1.3→V1.4）：新增內容或調整
- ZIP 檔名、資料夾名稱、所有頁面 footer 版本號三者保持一致
