## 如何上傳到 GitHub Pages

1. 前往 github.com 建立帳號，新建 repository（名稱如 study，選 Public）
2. 點「uploading an existing file」，把 zip 解壓縮後把所有檔案拖進去
3. 點「Commit changes」
4. Settings → Pages → Branch 選 main，資料夾選 / (root) → Save
5. 約 1 分鐘後：https://你的用戶名.github.io/study/
